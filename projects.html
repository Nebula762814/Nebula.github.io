nebula suite
homelab · 2024 – present
about

A cohesive set of self-hosted web applications running on a Mac Mini, all sharing a unified dark theme and served via Python HTTP servers managed by launchd on ports 7776–7778.

Includes a Plex media frontend with a custom player overlay and persistent credential storage, a FastAPI music server pointing at a local library, and a TUI-style notes app built with Python and Textual that syncs to GitHub.

2026main
robotics · FRC Team 2130 Alpha · 2026 season
about

Full robot software stack for Team 2130's 2026 FRC competition robot. Built in WPILib Java using a command-based architecture across six subsystems including swerve drive, launcher, and intake.

Integrated dual PhotonVision cameras for real-time AprilTag tracking and target alignment. Autonomous routines generated and tuned with PathPlannerLib. Driver-station telemetry and camera streams surfaced via Elastic Dashboard.

Reelbook
web · standalone html app · 2024
about

A single-file HTML sheet music reader designed for live performance use. Supports folder import, page navigation, and annotation tools, all without any server or install required.

Built with a dark interface to reduce eye strain on stage, and optimized for touch devices so it can run on a tablet or laptop in a live setting.

niri dotfiles
linux · CachyOS · Wayland · ongoing
about

Full compositor configuration for niri, a scrollable-tiling Wayland compositor, running on CachyOS (Arch-based). Includes complete keybind layout, per-app window rules, and hardware-specific fixes for touchpad, brightness keys, and keyboard backlight.

Themed consistently across the entire environment with a dark monospace aesthetic. Dotfiles are version-controlled and maintained as a living configuration.
